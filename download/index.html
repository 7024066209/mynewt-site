The next release of Mynewt is coming soon! The anticipated release date is March 21st. The release point is marked with the tag `mynewt_0_8_0_b2_tag`

If you are interested in trying out Mynewt for a project we recommend waiting until this upcoming release. It makes package management and build for testing simpler and more intuitive. It is for use by developers and interested community members only. The links to the download page for general consumption will be posted on the release date after the release is approved. Code in development ¶

While the use of one of the official releases listed above is generally recommended, you may be interested in seeing work in progress. The most recent code resides in the develop branch of the Mynewt git repository. You may access the code for Mynewt OS and Newt Tool from the Apache mirror on github.com.

  • Apache Mynewt OS mirror on github.com
  • Apache Newt Tool mirror on github.com
